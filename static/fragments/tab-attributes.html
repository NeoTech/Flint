Core Attributes

Attribute Description
hx-getIssues a GET request to the given URL
hx-postIssues a POST request to the given URL
hx-targetSpecifies the target element to swap content into
hx-swapControls how content is swapped (innerHTML, outerHTML, etc.)
hx-triggerSpecifies the event that triggers the request
hx-boostProgressively enhances links and forms
