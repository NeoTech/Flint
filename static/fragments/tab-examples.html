Quick Examples

<button hx-get="/page.html" hx-target="#content">Load</button>

Click a button → load static HTML → swap it into #content

<a hx-boost="true" href="/about">About</a>

Navigate without full page reload — SPA-like behavior for free

<div hx-get="/news.html" hx-trigger="revealed"></div>

Lazy-load content when the element scrolls into view
